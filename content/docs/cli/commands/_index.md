The commands in this section give you the full range of
Velociraptor's features from your terminal. Whether you are
running VQL queries, managing artifacts, mounting collection
containers, or configuring the server, the CLI gives you a
direct way to work with Velociraptor without the graphical
interface.

Browse the command groups listed below to find the documentation
for each command and its subcommands. You can also run
`velociraptor --help` from your terminal for a quick overview of
the available commands.
