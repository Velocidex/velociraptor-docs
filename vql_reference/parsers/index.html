    Parsers

    Many Velociraptor artifacts rely on parsing of file and data formats.

    Velociraptor has dedicated parsers for some specialized file and data formats. In addition we have flexible generic parsers, such as grok and other regex-based parsers for text formats, and parse_binary which provides completely customizable parsing of binary file formats.
